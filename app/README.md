# Learn Nuxt 3

This is my playground for learning nuxt and vuetify.

## Setup

### Install dependencies

```bash
# npm
npm install

# pnpm
pnpm install
```

### Start development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

Locally preview production build:

```bash
npm run preview
```
